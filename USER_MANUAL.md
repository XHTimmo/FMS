# 系统操作手册

本手册介绍收支管理与会员管理模块间的数据同步，以及新增的“撤销缴费”功能的操作方式。

## 1. 撤销会员缴费

### 功能说明
如果在记录会员缴费时发生错误（如金额错误、会员选错等），您可以在会员管理界面对已经缴纳的会费记录进行撤销。撤销后：
- 该缴费记录状态将变更为“已撤销”。
- 系统会自动查找并回滚（软删除）与该缴费记录关联的收支账单，并将账单金额从对应账户余额中扣除（或退回）。
- 撤销操作会被记录在系统的审计日志中，包含操作人、时间及原因，防止篡改。
- 撤销后，该记录将不再计入会员的最新有效期中，系统会自动重新计算会员状态（如恢复为逾期或冻结状态）。

### 操作步骤
1. 登录系统，进入 **会员管理** 页面。
2. 找到需要撤销缴费的会员，点击操作列的 **缴费历史** 按钮。
3. 在弹出的“缴费历史”窗口中，找到状态为“已缴”的错误记录。
4. 在“凭证管理”列下点击 **撤销** 按钮（黄色字样）。
5. 系统会弹出确认对话框并要求输入 **撤销原因**。
6. 输入有效原因后，点击 **确认撤销**。
7. 撤销成功后，记录状态将更新，并实时刷新会员列表状态。

## 2. 收支管理删除会员缴费账单

### 功能说明
作为双向数据一致性的一部分，财务人员在“收支管理”中也可以直接删除属于会员缴费生成的账单。

### 操作步骤
1. 进入 **收支管理** 页面。
2. 找到由“会员缴费”生成的入账账单。
3. 点击 **删除**，系统会弹出确认提示。
4. 确认删除后，对应账户余额会扣减，同时系统会自动将会员管理中对应的这笔缴费记录状态更新为“已撤销”。
5. 会员管理界面将实时更新该会员的最新缴费有效期和状态。

## 3. 注意事项
- **防止重复撤销**：系统内置防重机制，已经撤销的记录将无法再次被撤销。
- **权限控制**：撤销缴费和删除账单操作需要拥有 `admin` 或 `finance` 角色权限，普通员工无法执行该操作。
- **事务一致性**：上述数据同步过程在底层使用了数据库事务机制，如在同步过程中出现任何错误（如网络、磁盘故障等），所有数据将自动回滚，确保账单与缴费记录永远保持一致。
